## Getting Started

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This project used [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Getting Started

First, run the development server:

```bash
npm run dev
```

To run production server, npm lint must display any errors. Then:

```sh
npm run build
npm run start
```

Project defaults to [http://localhost:3000](http://localhost:3000)
